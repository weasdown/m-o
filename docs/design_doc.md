# Design Document

This document will give an overview of M-O's design.

## Functions

| 'Function' | Implementation |
| -------- | -------- |
| Propulsion | [Ballbot]-style spherical wheel |
| Head movement | Pan-tilt mount ([example mount]) |
| Arm movement | Stepper motors |
| Brush | Possibly 3D-printed TPU |
| Warning light | COTS rotating light or red LEDs mounted on a motor |
| Eyes | LCD screen |
| Blue backpack | 3D-printed backpack |
| Blue brush cover | 3D-printed brush cover |
| Brush cover movement | Stepper motor? |
